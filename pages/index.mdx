# Instrução de como instalar essa documentação no Vercel e Github
 
1. Baixar o VScode
2. Instalar: Live Preview, Git Lens
3. Crie sua conta no Github
4. Crie sua conta do Vercel e crie um git repository
5. Deploy este documento no Vercel e instale o no github: https://vercel.com/templates/next.js/documentation-starter-kit
5.1 Clique no botão go to dashboard e clique em repository
6. No github aperte o botão code e copie o código clone para o clipboard
7. Vá no VScode
7.1 abrir pasta, quando abrir uma pop-up, crie uma nova pasta e abrir
7.2 abra o terminal (menu: ver -> Terminal | ou vai no canto direito da tela e clique no segundo icone (da esquerda pra direita) que é um quadrado com divisória abaixo dele)
8. Digite no terminal: git clone, cole o código que você copiou e pressione enter
8.1 Caso de um xcode error, vc pode pesquisar no chatgpt ou abrir o terminal do computador (não do vscode), copiar o codigo que ele vai dar e instalar o xcode.
9. Retorne para o terminal do VScode e digite: rm -rf .git (isso vai limpar o vinculo com o codigo do outro dono – n vai dar nenhuma msg de retorno)
10. Ative salvamento automatico (menu -> arquivo -> salvamento automatico)
11. No menu vertical tem o nome da pasta que você está, abra a subpasta ***documentation*** que está lá dentro e abra a pasta ***pages*** -> index.mdx 
12. Escreva algo la dentro pra testar
13. Clique no menu lateral controle de codigo fonte (abaixo da lupa) vai aparecer um icone circular com número
14. Digite uma mensagem qualquer e aperte confirmação (se aparecer uma msg de erro apeee depois clique novamente no botão. (se você n colocar a msg e apertar o botão vai abrir um novo arquivo e vc pode digitar a msg nele, salvar e fechar esse aquivo e apertar novamente o botão azul)
14.1 Caso dê erro é pq vc precisa criar o .gitignore, digite no terminal ```git config --global user.name seunomeaqui```e depois ```git config --global user.email seuemailaqui``` (provavelmente qnd vc apertar commit vai abrir um arquivo, la linha 1 você escreve qquer coisa, salva e aperta commit)
15. Retorne no site do Vercel e acesse o link que ele criou


## Como rodar local?

1. no terminal escreva ```npm instal````
2. escreva ```npm run dev````
3. abra o navegador e digite
