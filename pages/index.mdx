# Introduction
 
1. Baixar o VScode
2. Instalar: Live Preview, Git Lens
3. Crie sua conta no Github
4. Crie sua conta do Vercel
5. Deploy este documento no Vercel e instale o no github: https://vercel.com/templates/next.js/documentation-starter-kit
6. No github aperte o botão clone, copie o código
7. Vá no VScode, abra o terminal (menu: ver -> Terminal)
8. Cole o código que você copiou
9. Digite rm -rf .git (isso vai limpar o vinculo com o codigo do outro dono)
10. Ative salvamento automatico (menu -> arquivo -> salvamento automatico)
11. Abra a pasta pages -> index.mdx 
12. Escreve as instruções
13. Clique no menu lateral controle de codigo fonte
14. Digite uma mensagem qualquer e aperte confirmação e depois clique novamente no botão
