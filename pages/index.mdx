# Instrução
 
1. Baixar o VScode
2. Instalar: Live Preview, Git Lens
3. Crie sua conta no Github
4. Crie sua conta do Vercel e crie um git repository
5. Deploy este documento no Vercel e instale o no github: https://vercel.com/templates/next.js/documentation-starter-kit
5.1 Clique no botão go to dashboard e clique em repository
6. No github aperte o botão code e copie o código clone para o clipboard
7. Vá no VScode
7.1 abrir pasta, quando abrir uma pop-up, crie uma nova pasta e abrir
7.2 abra o terminal (menu: ver -> Terminal)
8. Digite no terminal: git clone, cole o código que você copiou e pressione enter
8.1 Caso de um xcode error, vc pode pesquisar no chatgpt ou abrir o terminal do computador (não do vscode), copiar o codigo que ele vai dar e instalar o xcode.
9. Retorne para o terminal do VScode e digite: rm -rf .git (isso vai limpar o vinculo com o codigo do outro dono)
10. Ative salvamento automatico (menu -> arquivo -> salvamento automatico)
11. Abra a pasta pages -> index.mdx 
12. Escreve as instruções
13. Clique no menu lateral controle de codigo fonte (abaixo da lupa)
14. Digite uma mensagem qualquer e aperte confirmação e depois clique novamente no botão. (se você n colocar a msg e apertar o botão vai abrir um novo arquivo e vc pode digitar a msg nele, salvar e fechar e apertar novamente o botão azul)
15. Retorne no site do Vercel e acesse o link que ele criou
